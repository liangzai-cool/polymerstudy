# PolymerStudy
A simple for studying polymer.
# Dependencies
- [Node](https://nodejs.org/)

# Get Started
1.Clone Project
```shell
git clone git@github.com:liangzai-cool/polymerstudy.git
```
2.Install `polymer`, `bower`
```
npm install -g polymer bower
```
3.Install dependencies
```
bower install
```
4.Running
```
polymer serve --open
```
